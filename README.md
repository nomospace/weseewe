# 🎮 WeSeeWe - 跳跃方块 v2.0

> 踩对平台，跳得更高！纯 Canvas 实现的跳跃类小游戏

## 🎯 游戏玩法

- **点击屏幕**或**按空格键**跳跃（支持二段跳）
- 踩在不同类型的平台上获得不同分数
- 收集道具获得增益效果
- 掉出屏幕则游戏结束

## 🎨 平台类型

| 颜色 | 类型 | 效果 |
|------|------|------|
| 🟢 绿色 | 安全平台 | +5 分，稳定可靠 |
| 🔵 蓝色 | 普通平台 | +1 分 |
| 🟡 黄色 | 奖励平台 | +15 分 + 连击加成 |
| 🟣 紫色 | 弹射平台 | +3 分，超级弹跳 🚀 |
| 🔴 红色 | 危险平台 | ⚠️ 会碎裂！避免踩踏 |

## ✨ 道具系统

| 道具 | 效果 |
|------|------|
| 🛡️ 护盾 | 5秒内可踩危险平台 |
| 🧲 磁铁 | 3秒内自动吸引金币 |
| 2️⃣ 双倍 | 5秒内分数翻倍 |

## 🎮 特性

- 🎨 纯 Canvas 实现，无框架依赖
- 📱 支持移动端和桌面端
- 🎵 Web Audio 音效
- 🏆 本地最高分 + 排行榜
- 📈 渐进式难度系统
- ⚡ 轻量级，快速加载
- 🔄 复活机制

## 🚀 本地运行

```bash
# 克隆项目
git clone https://github.com/nomospace/weseewe.git
cd weseewe

# 使用任意 HTTP 服务器
python3 -m http.server 8080
# 或
npx serve .

# 访问 http://localhost:8080
```
